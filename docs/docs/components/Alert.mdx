The `<Alert />` component provides feedback to the user that something has happened.

It has the attribute `role="alert"`, which means that when the component is rendered its content will also be read aloud to users with screen readers.

The Alert component includes predefined status codes and customizable colors and icons.

```js
import { Alert } from 'minerva-ui';
```

If status codes are supplied, the component will supply predefined colors and icons.

```jsx
<>
  <Alert
    status="error"
    title="Whoa!"
    body="Something not great is happening."
  />
  <br />
  <Alert
    status="success"
    title="Congrats!"
    body="Something great is happening!"
  />
  <br />
  <Alert
    status="warning"
    body="Something is happening that isn't bad yet, but might be soon."
  />
  <br />
  <Alert
    status="info"
    title="Attention:"
    body="Something is happening and you should know about it."
  />
</>
```

## Alerts that close

You can supply a prop of `canBeClosed` to allow an Alert to display and hide.

```jsx live
() => {
  const [open, setOpen] = React.useState(false);

  return (
    <>
      {!open && <Button onClick={() => setOpen(true)}>Open Alert</Button>}
      <Alert
        canBeClosed
        closeText="Got it!"
        isOpen={open}
        onClose={() => setOpen(false)}
        status="error"
        title="Whoa!"
        body="Something not great is happening!"
      />
    </>
  );
};
```

## Custom Colors / Icons

You can also supply your own colors and icons. The default theme includes [Feather Icons](https://feathericons.com/).

```jsx live
<>
  <Alert icon="arrow-right-circle" bg="orange.200" title="Alert!!!" />
  <br />
  <Alert icon="circle" bg="green.100" body="Here is some information." />
</>
```

## Props

Props related to the Alert component.

| Name           | Type                                  | Is Required | Default | Description                                                                          |
| -------------- | ------------------------------------- | ----------- | ------- | ------------------------------------------------------------------------------------ |
| `status`       | 'error', 'success', 'warning', 'info' | optional    | none    | If status codes are supplied, the component will supply predefined colors and icons. |
| `title`        | string                                | optional    | none    | Title for Alert                                                                      |
| `body`         | string                                | optional    | none    | Body for Alert                                                                       |
| `bg`           | string                                | optional    | #e5e7eb | Sets the background color of the alert. Will override `status`                       |
| `icon`         | string                                | optional    | none    | Icon for Alert                                                                       |
| `isOpen`       | boolean                               | optional    | true    | Sets if Alert is open                                                                |
| `canBeClosed`  | boolean                               | optional    | false   | Sets is Alert can be opened and closed                                               |
| `onClose`      | function                              | optional    | none    | Action to take when alert is closed                                                  |
| `hasCloseIcon` | boolean                               | optional    | false   | Sets if Alert close button has an X icon                                             |
| `closeText`    | string                                | optional    | none    | Sets text for Alert close button                                                     |
