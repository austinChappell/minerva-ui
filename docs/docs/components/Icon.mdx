title: 'Icon'

import { Icon } from 'minerva-ui';

Basic icon component. SVGs can be added to the `icons` prop in your theme.

The default theme includes [Feather Icons](https://feathericons.com/).

```js
import { Icon } from 'minerva-ui';
```

```jsx live
<Icon name="activity" />
```

## Color

```jsx live
<>
  <Icon name="plus" color="blue.600" />
  <Icon name="activity" color="purple.500" />
</>
```

## Size live

```jsx
<>
  <Icon name="plus" size="16px" />
  <Icon name="activity" size="32px" />
</>
```
