The `<Link />` component extends the `<a>` tag.

It includes the prop `isExternal`, which will add the `target="blank"` and `rel="noopener noreferrer"` attributes, which are helps prevent [cross-origin security issues](https://developers.google.com/web/tools/lighthouse/audits/noopener) when opening a new page.

Insired by [Chakra UI's Link component](https://chakra-ui.com/link)

```js
import { Link } from 'minerva-ui';
```

## External Link

```jsx live
<Link href="https://www.github.com/cratebind/minerva-ui" isExternal>
  Minerva UI GitHub Page
</Link>
```

## Props

These are props related to the Link component.

|Name|Type|Is Required|Default|Description|
| --- | --- | --- | --- | --- |
| `href` | string | optional| none | Url for link |
| `isExternal` | boolean | optional| none | Opens link in new tab |
