Accessibility-focused tabs. Built on top of the [Reach UI Tabs](https://reacttraining.com/reach-ui/tabs).

```js
import {
  Box, 
  Flex,
  Tabs,
  TabList,
  Tab,
  TabPanels,
  TabPanel,
  useTabContext,
} from 'minerva-ui';
```

## Tabs with underline

```jsx
<Tabs>
  <TabList mb="20px">
    <Tab mr="15px">My Account</Tab>
    <Tab mr="15px">Favorites</Tab>
    <Tab mr="15px">Orders</Tab>
    <Tab>Billing</Tab>
  </TabList>
  <TabPanels>
    <TabPanel>
      <p>My Account!</p>
    </TabPanel>
    <TabPanel>
      <p>Favorites!</p>
    </TabPanel>
    <TabPanel>
      <p>Orders!</p>
    </TabPanel>
    <TabPanel>
      <p>Billing!</p>
    </TabPanel>
  </TabPanels>
</Tabs>
```

## Props

These are props related to the Skeleton component.

| Name             | Type    | Is Required | Default | Description                                   |     
| ---------------- | ------- | ----------- | ------- | ----------------------------------------------|
| `tabListBgColor` | string  | optional    | #fff    | Background Color of TabList                   |
| `tabColor`       | string  | optional    | none    | Color of Tab                                  |
| `activeBg`       | string  | optional    | none    | Background Color of active Tab                |
| `activeColor`    | string  | optional    | none    | Color of active Tab                           |
| `activeWeight`   | string  | optional    | none    | Font Weight of active Tab                     |
| `underline`      | boolean | optional    | true    | Toggles underline visibility below active Tab |
| `underlineColor` | string  | optional    | none    | Color of underline below active Tab           |
| `underlineSize`  | string  | optional    | 1px     | Size in px of underline below active Tab      |
| `pills`          | boolean | optional    | false   | Toggles pill style active Tab                 |
