import { Text } from 'minerva-ui';

Basic text component, replaces `<p>`. Uses [Tailwind CSS font size](https://tailwindcss.com/docs/font-size) scaling.

```js
import { Text } from 'minerva-ui';
```

```jsx live
<Text>Text</Text>
```

## Text Font Sizes

```jsx live
<Stack>
  <Text fontSize="xs">Extra Small Text</Text>
  <Text fontSize="sm">Small Text</Text>
  <Text fontSize="md">Medium Text</Text>
  <Text fontSize="lg">Large Text</Text>
  <Text fontSize="xl">Extra Large Text</Text>
  <Text fontSize="2xl">2x Large Text</Text>
</Stack>
```


## Props

These are props related to the Text component.

| Name         | Type    | Is Required | Default | Description           |
| ------------ | ------- | ----------- | ------- | --------------------- |
| `lineHeight` | string | optional    | normal | Sets line height of font |
