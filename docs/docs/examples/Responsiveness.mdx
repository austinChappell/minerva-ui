## Responsive Design

"Often when working on responsive layouts, it's useful to adjust styles along a singular dimension â€“ such as font-size, margin, padding, and width. Instead of manually managing media queries and adding nested style objects throughout a code base, Styled System offers a convenient shorthand syntax for adding responsive styles with a mobile-first approach."  
[Styled System](https://styled-system.com/responsive-styles/)  
[Array Props](https://styled-system.com/guides/array-props)

```jsx live
// Re-size your browser to view changes
<Text fontSize={[36, 24, 18]}>Hello, World</Text>
```

Alternatively, you can define breakpoints with aliases and use plain objects as values. Any undefined alias key will define the base, non-responsive value.

```
    // theme.js
    const breakpoints = ['640px', '768px', '1024px', '1280px'];

    // aliases
    breakpoints.sm = breakpoints[0];
    breakpoints.md = breakpoints[1];
    breakpoints.lg = breakpoints[2];
    breakpoints.xl = breakpoints[3];

    export default {
        breakpoints,
    };
```

```jsx live
// Re-size your browser to view changes
<Flex
  alignItems="center"
  justifyContent="center"
  borderRadius={5}
  backgroundColor="blue.300"
  height={100}
  width={{ sm: '100%', md: 1 / 2, lg: 1 / 3, xl: 1 / 4 }}
  p={[2, 3, 4]}
>
  <Text fontSize={[36, 24, 18]}>Hello, World</Text>
</Flex>
```
