---
title: 'Spinner'
---

The `<Spinner />` component renders a basic loading spinner made with CSS.

```js
import { Spinner } from 'minerva-ui';
```

```jsx live
<Spinner />
```

## Props

These are props related to the Spinner component.

| Name         | Type                          | Is Required | Default | Description                                                                                                    |
| ------------ | ----------------------------- | ----------- | ------- | -------------------------------------------------------------------------------------------------------------- |
| `size`       | 'xs', 'sm', 'md', 'lg', 'xl', | optional    | none    | Size of the spinner                                                                                            |
| `emptyColor` | string                        | optional    | none    | Color of the empty area in the spinner                                                                         |
| `color`      | string                        | optional    | none    | Color of the spinner                                                                                           |
| `thickness`  | string                        | optional    | none    | Thickness of the spinner                                                                                       |
| `speed`      | string                        | optional    | none    | Speed of the spinner                                                                                           |
| `label`      | string                        | optional    | none    | For accessibility, it's important to add a fallback loading text. This text will be visible to screen readers. |
