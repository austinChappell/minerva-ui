---
title: 'Heading'
---

Basic heading component, replaces `<h1> <h2> <h3> <h4>`

```js
import { Heading } from 'minerva-ui';
```

```jsx live
<Heading>Heading</Heading>
```

# Variants

### h3 Heading

```jsx live
<Heading as="h3">H3 Heading</Heading>
```

### Small h4 Heading

```jsx live
<Heading as="h4" size="sm">
  Heading
</Heading>
```
