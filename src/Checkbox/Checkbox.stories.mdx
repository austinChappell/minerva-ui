# Checkbox

Extends `<input type="checkbox" />`, allows styling input

```js
import { Checkbox } from 'minerva-ui'
```

```jsx
() => {
  const [checked, setChecked] = React.useState(false);
  return (
    <Checkbox
      checked={checked}
      onChange={() => setChecked(!checked)}
    >
      Stay Logged In
    </Checkbox>
  )
}
```
