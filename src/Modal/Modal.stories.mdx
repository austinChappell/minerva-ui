# Modal

Basic Modal component with predefined status codes and customizable color and icon.

```js
import { Modal } from 'minerva-ui'
```

If status codes are supplied, the component will supply predefined colors and icons.

```jsx
<Modal>
Hello World!
</Modal>
```
