# Modal

Basic Modal component that accepts an `isOpen` prop.

```js
import { Modal, ModalHeader, ModalBody, ModalFooter } from 'minerva-ui'
```

                </ModalBody>
                <ModalFooter>
                    <Flex px={6} py={3} flexDirection={['column', 'row-reverse']}>
