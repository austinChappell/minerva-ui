# Skeleton

Animated loading skeleton that automatically adapts to your app.

```js
import { Text } from 'minerva-ui';
```

```jsx
() => {
  const [data, setData] = React.useState('');
  return data ? <Text>test</Text> : <Skeleton />;
};
```


```jsx
() => {
  const [data, setData] = React.useState('');
  return data ? <Icon name="plus"></Icon> : <Skeleton />;
};
```

```jsx
() => {
  const [data, setData] = React.useState('');
  return data ? <Box border="2px solid red" borderRadius="full">Test</Box> : <Skeleton />;
};
```
