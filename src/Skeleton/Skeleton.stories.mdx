# Skeleton

Animated loading skeleton that automatically adapts to your app.

```js
import { Text } from 'minerva-ui';
```

```jsx
() => {
  const [data, setData] = React.useState('');
  return <div>{data ? <Text>test</Text> : <Skeleton />}</div>;
};
```

### Circular loading skeleton

```jsx
() =>  <Skeleton height="100px" width="100px" borderRadius="full" />;
```

### Three-line loading skeleton with custom gap and width. Gap defaults to 10px

```jsx
() => {
  const [data, setData] = React.useState('');
  return (
    <div>
      {data ? (
        <Text>test</Text>
      ) : (
        <Skeleton count={3} width="500px" gap="15px" />
      )}
    </div>
  );
};
```

### Use the button to toggle between the 'loaded' and 'loading' states of the image

```jsx
() => {
  const [loading, setLoading] = React.useState(false);
  return (
    <>
      <Button mb="10px" onClick={() => setLoading(prevState => !prevState)}>
        {loading ? 'Loading...' : 'Loaded'}
      </Button>
      <div>
        {!loading ? (
          <Image
            src="https://source.unsplash.com/nc8Qwfie-tU/400x400"
            height="200px"
            width="200px"
          />
        ) : (
          <Skeleton width="200px" height="200px" />
        )}
      </div>
    </>
  );
};
```
