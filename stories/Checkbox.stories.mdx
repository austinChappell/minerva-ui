# Checkbox

Generic wrapper for a style-able checkbox input

```js
import { Checkbox } from 'minerva-ui'
```

```jsx
() => {
  const [checked, setChecked] = React.useState(false);
  return (
    <ThemeProvider>
      <Checkbox
        checked={checked}
        onChange={() => setChecked(!checked)}
      >
        Stay Logged In
      </Checkbox>
    </ThemeProvider>
  )
}
```
