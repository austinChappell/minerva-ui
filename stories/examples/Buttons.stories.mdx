# Buttons


### Button Group
```jsx isManual=true
const CustomButton = (props) => (
  <Button bg="gray.300" borderWidth={0} {...props} />
);

render(
  <Flex>
    <CustomButton borderTopRightRadius={0} borderBottomRightRadius={0}>Button 1</CustomButton>
    <CustomButton borderTopLeftRadius={0} borderBottomLeftRadius={0}>Button 2</CustomButton>
  </Flex>
)
```

### Button Hover Styles
```jsx
<Button
  _hover={{
    backgroundColor: '#551A8B',
    color: '#fff',
    borderColor: '#551A8B'
  }}
>
  Hover Button
</Button>
```
